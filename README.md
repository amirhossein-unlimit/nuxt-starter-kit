## راه‌اندازی پروژه

1. کلون گرفتن
   ```bash
   git clone https://github.com/amirhossein-unlimit/university
   ```
2. نصب وابستگی ها
   ```bash
   pnpm install
   ```
3. اجرای پروژه در حالت توسعه
   ```bash
   pnpm dev
   ```
4. ساخت پروژه
   ```bash
   pnpm build
   ```
5. پیش‌نمایش نهایی پروژه
   ```bash
   pnpm run preview
   ```
